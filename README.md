**Creation d'un projet Maven avec Hibernate et H2**

Ce projet montre comment utiliser de JPA avec Hibernate pour effectuer des opérations de persistance sur une base de données en mémoire. En démondrans:
le mapping objet-relationnel
-la gestion des transactions
-l’insertion et la lecture de données
-l’utilisation de JPQL
